自动发送 token

针对 signup 和 login 做脚本配置，每次登陆自动将 jwt 返回的 token 存在 environment 中，在获取 api 时使用该环境变量，自动获取登录权限

  1. 配置脚本写入环境变量

  1. 运行 signup/login，查看环境变量是否生成

  2. 没有 token 的话，获取数据会报错

  3. 在一个受保护的 api 中，输入环境变量，点击 send 就会正常 get 数据了

  4. 可以看到，header 自动填充了 token 数据
